## 🚀 Getting Started

dbdb-index is a Windows app that helps you view a large catalog of database systems in one place. It lists 1071 systems and includes stats, charts, and full metadata from dbdb.io.

Use it to search, compare, and review database systems without opening many sites.

## 🛠️ Common Issues

### The app will not open
- Make sure the download finished
- Check that you extracted the ZIP file
- Right-click the app and choose Run as administrator

### Windows blocks the file
- Open the release page again and check you downloaded the latest file
- If Windows shows a security prompt, choose the option to run the app

### The window looks too small
- Increase display scaling in Windows
- Maximize the app window
- Lower your browser zoom if you opened the release page in a browser window

### Search returns no results
- Try a shorter name
- Check spelling
- Use a known vendor or system family name
